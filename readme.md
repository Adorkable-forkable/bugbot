## Dev setup

1. Clone this repo and run `npm i`.
2. Create a `.env` file with:
  - `GITHUB_CLIENT_ID`
  - `GITHUB_CLIENT_SECRET`
  - `SLACK_CLIENT_ID`
  - `SLACK_CLIENT_SECRET`
  - `AWS_REGION` (if you are using the dynamo adapter)

### Run the tests

    npm test

### Start the server

    npm start
